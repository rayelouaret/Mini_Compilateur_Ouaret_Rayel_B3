Projet Mini-Compilateur - OUARET RAYEL B3


Contenu du dossier :
- MiniCompilateurJS.jar : L'executable Java.
- test.txt : Le code source a analyser (modifiable).
- LANCEUR.bat : Script pour lancer le programme facilement.

INSTRUCTIONS D'EXECUTION :
Option 1 (Recommandee) :
Double-cliquez simplement sur le fichier "LANCEUR.bat". Une fenetre s'ouvrira avec les resultats.

Option 2 (Manuelle) :
Ouvrez une invite de commande dans ce dossier et tapez :
java -jar MiniCompilateurJS.jar
